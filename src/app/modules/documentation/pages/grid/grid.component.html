Bootstrap Grid System

This section provides an in-depth look at the Bootstrap Grid System and how to use it effectively to create responsive layouts.

Introduction

Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content. It's built on a flexible 12-column system that scales to accommodate screens of all sizes from small phones to large desktops.

Basic Grid Layout

.col-sm-4
.col-sm-4
.col-sm-4

This setup divides the row into three equal columns on small (sm) and larger devices.

Responsive Adjustments

.col-12 .col-md-8
.col-6 .col-md-4

Columns adjust for different breakpoints; full width on extra small devices, and split on medium (md) or larger screens.

Gaps

Gaps (gutters) can be adjusted using 'g-*' classes, where '*' is the gap size. Here, 'g-3' adds more space between columns.
